Things to do after Linux Mint 17 Installation

Nov 16, 2015

Using Linux is like driving a manual transmission vehicle, wherein you have complete control on most of the aspects of the system. This post contains various tips & tricks to configure your linux machine.

  1. Change keyboard layout to use Num Key as in windows
    (Keyboard->Layout->Options->Miscellenous->Windows)
  2. Delete Downloads, Music, Video folders, etc and Drag and Drop same Folders from Storage Drive to the side pane.
  3. If firefox doesn't support Google, add an Addon. Google Default
  4. Fix booting issues by installing boot-repair. Boot-Repair - Community Help Wiki
  5. Install Chrome. Download chrome from website
  6. Set Download locations of chrome and firefox to Storage->Downloads->Chrome
  7. Install Geany IDE.
  8. Install libraries for C compilation.
  9. If brightness buttons don't work. Fix Brightness Control Not Working for Ubuntu 14.04 & Linux Mint 17
  10. Install atom text editor. Install Atom Text Editor In Ubuntu 14.04 & Linux Mint 17
  11. Install Oracle Java. How to Completely Install Oracle JDK on Linux Mint 17
  12. Applets in Panel
    1. Screenshot screenshot@tech71
    2. Multicore System Monitor multicore-sys-monitor@ccadeptic23
    3. Shutdown applet shutdownapplet@shelley
    4. System monitor with graphs system-monitor@pixunIII
    5. pomedoro
  13. To prettify GRUB Make your Grub boot menu pretty - Easy Linux tips project
  14. To auto mount Drives
  15. Set Mint to use local time.(keeps windows and ubuntu time sync)
  16. To install latex How to: Get LaTeX running on Linux Mint.
    Install the mentioned packages using a package manager.
  17. Use Evernote in Linux by NeighborNote  NeighborNote
  18. [experimental] Use symbolic links & Dropbox to keep important stuff in sync between windows, linux & cloud. How to Sync Any Folder to the Cloud With Symbolic Links
  19. Have bluetooth turned off at startup
  20. Set brightness to optimum level on startup Force set display brightness
