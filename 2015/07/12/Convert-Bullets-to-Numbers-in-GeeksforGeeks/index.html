Jul 12, 2015
  1. Install Greasemonkey @Greasemonkey for Firefox or Tampermonkey @Tampermonkey for Chrome or GreaseKit @GreaseKit for Mac for Safari.

  2. If using firefox, make sure you Restart your browser, before continuing further.

  3. Go to this page and click on Raw to install the script xerosanyam/GeeksForGeeks Bullets to Numbers.user.js
