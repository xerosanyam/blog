Pushing Android Project to Github

Oct 30, 2015

  1. Add icon to project if not added already, take use of following website; Android Asset Studio - Icon Generator - Launcher icons

  2. VCS > Import into Version Control > Create Git Repository > Select complete folder {Important}

  3. VCS > Import into Version Control > Share Project on Github

  4. Add README.md

  5. Extract app using App Extractor, push to drive, shorten url.

  6. Take screenshot. Post screenshot to a github issue to upload on github server. {Ultimate Hack}
  7. In README, describe project, give link to project, add screenshot {using img html tag and attribute width}, add license info.
