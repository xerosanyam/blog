Forwarding port in Linux

Mar 26, 2015

This forwards request from 8080 to 3000

sudo iptables -t nat -A PREROUTING -p tcp –dport 8080 -j REDIRECT –to-port 3000

This saves the iptables settings so that configurations don't reset after reboot.

sudo service iptables save
