Genymotion; (Emulator for Android)

Sep 18, 2015

After installing genymotion, when I tried to build{download} a virtual drive, it gave me file corrupt error. Following is the solution I found:
1. Remove all virtual box installation before installing genymotion.
2. Remove all files from appdata\genymotion\ folder
3. Download genymotion virtual drive externally. { Will update the link soon}
4. Copy this to appdata\genymotion~\ova\  folder.
5. Install genymotion.
6. Add virtual device, choose the same device as you had downloaded in step 3.
7. It will automatically detect the device.
8. Start the device.
9. If may give some error {happens almost everytime}, start device from virtual box.
10. It may also give error, close device. Start again. Again error, then switch off the device.
11. Now, start device from genymotion.
12. I have to follow steps 8-11 everytime I start device from genymotion, since it is fast, and provides multiple features, I feel it is worth it.

If you happen to find any emulator better than genymotion, please comment. :)
