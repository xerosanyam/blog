Making Geany more powerful; Settings for Geany

Jun 28, 2015

Geany is a lightweight, fully customizable IDE that I use on linux. It makes coding a breeze. :)

  1. Insert date and time. Default Key: Shift+Alt+Delete. Set custom date format as : //#%d %b, %a; %I:%M %p. This prints 28 Jun, Sun; 4:30 PM. For more options see http://www.cplusplus.com/referenâ€¦

  2. Document->Use auto-indentation, Line Wrapping

  3. View->Show markers margin, show line numbers, show white spaces.

  4. Preferences-> Comment Alt+C, Uncomment Alt+V.
  5. Preferences->various->compiler tab autoscroll(NO), messages(n),scribble(n),status(n)

  6. Set Debug and Execute Debugging commands in Build Menu->Set Build Commands. Ensure you write separately for gcc and g++.

  7. Set build shortcut as F9, not Compile's options.
