Playing with Free BSD

Nov 27, 2015
  1. BSD doesn’t require sharing of code.
  2. Not controlled by one user.(Linux is controlled by Linus Trovalds)
  3. Maintains entire OS not just Kernel
  4. “BSD is what you get when a bunch of UNIX hackers sit down to try to port a UNIX system to the PC. Linux is what you get when a bunch of PC hackers sit down and try to write a UNIX system for the PC.” Therefore, BSD is much similar to UNIX.

Java on Linux

Nov 04, 2015

  1. Set PATH: used to run javac & other bin applications from anywhere. Otherwise you need to give the complete path. (Was set automatically, somehow)

  2. Set CLASSPATH: used to look for jdk tools & user classes. You won't be able to use any class from current directory if you do no set it.In Java it is important to know from where you are compiling your program.
